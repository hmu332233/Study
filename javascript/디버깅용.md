## 디버깅용

### trace
이 함수가 호출되기까지의 callstack을 출력해준다.
```js
console.trace();
```

### group
출력되는 내용을 그룹지어서 출력해준다.
```js
console.group();
console.log("A");
console.log("B");
console.log("C");
console.groupEnd();

console.groupCollapsed("groupA");
console.log("A");
console.log("B");
console.log("C");
console.groupEnd();
```


### 노드 가져오기
```js
$(id) : id에 해당하는 엘리먼트
$$(selector) : selector에 해당하는 엘리먼트 배열
$x(xpath) : xpath 구문
```
