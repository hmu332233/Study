## github 브런치 가져오기


- #### 브런치 목록
  - `git branch` 로컬저장소 브런치 목록
  - `git branch -r` 원격저장소 브런치 목록
  - `git branch -a` 모든 브런치 리스트

- #### 브런치 가져오기
  - `git checkout -b 생성할브런치이름 원격브랜치이름`  
    ex) `git checkout -b develop origin/develop`  
    <br/>
  - `git checkout -t origin/develop` 위와 같은 기능을 한다.
