# Template Method

### 개요

![1](/assets/template/main.PNG)

- 상위 클래스에서 **처리의 흐름을 제어**하며, 하위클래스에서 **처리의 내용을 구체화** 한다.

- 여러 클래스에 공통되는 사항은 상위 추상 클래스에서 구현하고, 각각의 상세부분은 하위 클래스에서 구현한다.

- 코드 중복을 줄이고 리팩토링에 유리한 패턴으로 상속을 통한 확장 개발방법으로써 가장 많이 사용되는 패턴중에 하나이다.
