```
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```
- **media query**
  - if/else 같은 느낌으로 생각하면 쉽다
```
@media screen and (max-width: 768px) {..}
@media screen and (min-width: 769px) and (max-width: 991px) {..}
```
